# Ricarte

Proyecto con estructura de ramas:
- **main**: Rama estable (producción)
- **diseño**: Interfaz y UI/UX
- **backend**: API y lógica del servidor
- **frontend**: Interfaz de usuario
- **pruebas**: Testing y QA

## Estructura del proyecto

```
Ricarte/
├── diseño/
├── backend/
├── frontend/
└── pruebas/
```

## Flujo de trabajo

1. Cada integrante trabaja en su rama
2. Hace commits y push a su rama
3. Crea Pull Request cuando termina
4. Se revisa y se hace merge a main
